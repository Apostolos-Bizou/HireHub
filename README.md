# HireHub

**The Professional Network for Maritime Crew**

LinkedIn for the maritime industry. Shipowners search a verified talent database, build shortlists, and forward candidates to their DMW-licensed manning agents for deployment.

## Project Status
- **Phase 1** (Foundation & IA): DONE
- **Phase 2** (UX/UI Design): DONE
- **Phase 3** (Prototype & API): IN PROGRESS

## Structure
- `docs/legal/` — Consent agreement, legal compliance assessment
- `docs/reports/` — Deliverables report, visual report, project tracker
- `database/` — Schema, migrations, seeds
- `api/` — API specifications
- `prototype/` — React interactive prototype
